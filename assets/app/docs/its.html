/it

API Documentation to It (invested time) entity, an it is period of time allocated by user performing any registered activity.

Fields

The IT entity has the fields:

  • id {number} - the It identifier on database
  • duration {number} - the time in minutes invested performing the task
  • createdAt {date} - the it creation date
  • investedTimeAt {number} - Indicating the activity id

Url Parameters

The server accept the following parameters in the route

  • limit - The maximum number of results to retrieve
  • skip - The total of resource to skip into database
  • sort - The sort type to apply in the resources (ASC or DESC)
  • investedTimeAt - Used to pass the Activity Id to filter It

4.0 Create a new It (Invested Time)

Request route (Warning: InvestedTimeAt is requered , represents the Activity Id) 

 POST /it

 data { duration : 20 , investedTimeAt : 1 }

5.0 Updating an It (Invested Time)

  "message": "It updated successfully."
